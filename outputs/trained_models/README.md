# Trained models

Saved model files created when you run training scripts from `models/`.

**Examples:** `random_forest.joblib`, `xgboost.joblib`, `logistic_regression.joblib`, `gradient_boosting.joblib`

The Streamlit app in `frontend/` can load a chosen file from here for predictions. Large binary files are often gitignored; document which teammate produces which artifact in the team report.
