# Data

This folder holds all datasets used by the project.

- **`raw/`** — Original files as downloaded (e.g. Kaggle `Train_Dataset.csv`). Do not overwrite; treat as read-only for reproducibility.
- **`processed/`** — Cleaned, merged, or feature-engineered tables written by your preprocessing pipeline. Scripts in `models/` typically read from `raw/` first; point them here if you add a preprocessing step.

See each subfolder’s `README.md` for details.
